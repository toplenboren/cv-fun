# Hi there

## Install

`pip install -r requirements.txt`

## Run

`python main.py`

> You should approve all rights to camera, or the game wont work!

### Contols:

1. SPACE - pause
2. R - restart
3. Use your finger near the camera to control the paddle

## Playgrounds:

1. Game: 

`python game.py`

2. Detection:

`python cv.py`
